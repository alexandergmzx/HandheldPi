# HHT sound cues

These are original, synthesized workflow cues generated by:

```bash
scripts/make_sound_cues.py
```

They are deliberately short, low-level, 44.1 kHz 16-bit stereo WAV files. Both
channels contain the same signal because the GamePi20 board consumes only the
GPIO18 PWM channel. Keep the physical volume potentiometer low for first bring-up.

Cue names are the stable interface between `PickingStateMachine` and the audio
backend; regenerate all files after changing `scripts/make_sound_cues.py`.
